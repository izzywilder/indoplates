üîå API Integration Endpoints

RESTful API for third-party integration (dealers, SAMSAT, partners)

User & Application APIs

POST /api/v2/applications
Submit new plate application with documents
GET /api/v2/applications/{id}
Get application status and details
PUT /api/v2/applications/{id}
Update application (SAMSAT admin only)
POST /api/v2/verify/document
AI-powered document verification (KTP/STNK/SIM)

SAMSAT Integration APIs

POST /api/v2/samsat/submit
Submit application to SAMSAT for approval
POST /api/v2/samsat/approve
SAMSAT approval with digital signature
GET /api/v2/samsat/queue
Get pending applications in review queue

Blockchain & Certificate APIs

POST /api/v2/blockchain/certificate
Generate blockchain certificate
GET /api/v2/blockchain/verify/{hash}
Verify certificate authenticity via blockchain

Payment & Financial APIs

POST /api/v2/payment/process
Process payment (GoPay, OVO, DANA, Bank Transfer)
GET /api/v2/payment/reconcile
Daily payment reconciliation for government

Analytics & Reporting APIs

GET /api/v2/analytics/dashboard
Real-time dashboard metrics for SAMSAT admin
GET /api/v2/reports/generate
Generate compliance and audit reports
